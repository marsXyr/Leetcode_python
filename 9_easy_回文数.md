# Leetcode 9 回文数
***
### 题目描述

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

**示例1:**  

	输入: 121
	输出: true

**示例2:**  

	输入: -121
	输出: false
	解释: 从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
	
**示例3:**  

	输入: 10
	输出: false
	解释: 从右向左读, 为 01 。因此它不是一个回文数。
	
**进阶：**  
 
你能不将整数转为字符串来解决这个问题吗？


### 考点

数学


### 代码
执行用时: **100ms**, 内存消耗: **13.3MB**。

```
class Solution:
    def isPalindrome(self, x: int) -> bool:
        return str(x) == str(x)[::-1]
```
